# zkui简介：

网页版zookepper管理工具

# 主要功能：

添加、删除、导入、导出，zookepper节点数据，并支持Jasypt加密

# 后台框架框架：

SpringBoot 2.1.3

# 前端框架：

layui-v2.4.5

# 前后台交互方式：

前后台使用webSocket链接，所有节点内容以消息方式传输（JSON）

# 发布历史：

### v1.0.1

- 与zookepper建立长连接
- 添加、删除、导入、导出、修改节点

### v1.0.2

- 修改节点值增加Jasypt加密方式
